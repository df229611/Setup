Для новичков пользователей Ubuntu Linux, которые при установки указали одно имя ПК и хотят сменить его на другое. Для просмотра и изменения именb сервера (компьютера) используется команда hostname. Обычно имя системы хранится в файле /etc/hostname, который используется в процессе загрузки. Откройте окно терминала и введите следующие команды.

Просмотреть текущее имя сервера

hostname

Изменения имени сервера, ПК на Ubuntu

Процедура смены имени компьютера:

1. Введите следующую команду для редактирования файла /etc/hostname vi или nano текстовый редактор:

sudo nano /etc/hostname

Удалите старое имя и введите новое.

2. Далее отредактируйте файл /etc/hosts:

sudo nano /etc/hosts

Замените во всех строках старое имя на новое.

3. Перезагрузите систему для применения изменений:

sudo reboot
