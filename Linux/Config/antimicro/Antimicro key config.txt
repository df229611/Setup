1 Набор

Левый стик
Влево  = Мышь Влево
Вправо = Мышь Вправо
Вверх  = Мышь Вверх
Вниз   = Мышь Вниз

Правый стик
Влево  = Мышь Влево
Вправо = Мышь Вправо
Вверх  = Мышь Вверх
Вниз   = Мышь Вниз

L Trigger = Мышь LB (Левая кнопка мышки)
R Trigger = Мышь RB (Правая кнопка мышки)

Крестовина
Влево  = Кнопка Left
Вправо = Кнопка Right
Вверх  = Кнопка Up
Вниз   = Кнопка Down

LB = Backspace
LT = ESC
RB = Pg Up
RT = Pg Down

Back  ->Set 4 WH
Start ->Set 8 WH

X = SHIFT_L
Y  ->Set 5 WH
A  ->Set 3 WH
B = ALT_L
